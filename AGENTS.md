# AGENTS

- Stack do projeto: Phaser 3, TypeScript, Vite, HTML/CSS e build Web estática.
- Comandos principais: `npm install`, `npm run dev`, `npm run build`, `npm run preview`.
- Não usar backend.
- Não usar assets gerados por IA.
- Preferir formas geométricas criadas via código no Phaser.
- Manter o escopo pequeno: MVP funcional antes de qualquer polimento.
- Sempre rodar `npm run build` antes de considerar a tarefa pronta.
- Antes de mudanças grandes, consultar `documentacao_color_shift_codex.md` quando o arquivo estiver disponível.
- Para refatorações visuais, consultar `identidade_visual_color_shift.md` quando o arquivo estiver disponível.
- A imagem de conceito pode ser usada apenas como referência/moodboard, nunca como asset do jogo.
