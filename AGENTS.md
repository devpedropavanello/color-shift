# AGENTS

- Stack do projeto: Phaser 3, TypeScript, Vite, HTML/CSS e build Web estatica.
- Comandos principais: `npm install`, `npm run dev`, `npm run build`, `npm run preview`.
- Nao usar backend.
- Nao usar assets gerados por IA.
- Preferir formas geometricas criadas via codigo no Phaser.
- Manter o escopo pequeno: MVP funcional antes de qualquer polimento.
- Sempre rodar `npm run build` antes de considerar a tarefa pronta.
- Antes de mudancas grandes, consultar `documentacao_color_shift_codex.md` quando o arquivo estiver disponivel na raiz ou anexado ao projeto.
